# Consumo de APIs

Aplicación web desarrollada en HTML, CSS y JavaScript que consume APIs externas para obtener información dinámica de usuarios y generar avatares personalizados.

## Funcionalidades

- Consumo de API de usuarios (JSONPlaceholder)
- Generación de avatares mediante API DiceBear
- Selección aleatoria de usuarios
- Visualización de datos en tablas
- Generación de tarjeta visual con información integrada
- Manejo de errores en la carga de datos

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript (Fetch API)

## Uso

1. Abrir el archivo `index.html` en el navegador
2. Presionar el botón "Cargar Información"
3. Visualizar los datos obtenidos de las APIs
4. Generar la tarjeta del usuario

## Descripción

El sistema consume dos APIs externas: una para obtener datos de usuarios y otra para generar imágenes dinámicas. Se implementa programación asíncrona para manejar las solicitudes y mostrar la información en tiempo real.
